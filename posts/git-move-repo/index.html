Git Move Repo

Feb 13, 2020 17:38 · 60 words · 1 minute read git

  1. Clone the project from old repo url

    git clone <old_repo_url>
    1. Pull all branches
    git branch -r | grep -v '\->' | while read remote; do git branch --track "${remote#origin/}" "$remote"; done
git fetch --all
git pull --all

  2. Change remote to new repo url

    git remote set-url origin <new_repo>

  3. Push all to new repo url

    git push --all
git push --tags
