Golang Version Switch

Jan 18, 2023 10:09 · 54 words · 1 minute read golang

  1. remove /usr/local/go if you install golang using pkg installer
sudo rm -rf /usr/local/go
sudo rm /etc/paths.d/go
  1. use homebrew to install golang
brew install go
  1. use homebrew to install golang different versions
brew install go@1.19
brew install go@1.18
  1. switch between golang version, for example: switch from 1.18 to 1.19
brew unlink go
brew link go@1.19
