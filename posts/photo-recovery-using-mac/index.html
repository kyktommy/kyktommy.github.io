Photo Recovery Using Mac

Jun 15, 2022

Today my colleagues’ SD card is corrupted. All photos are gone. Here is my steps to recover photos, videos using mac.

  1. Plug the SD card in your mac, and choose ignore the corrupted sd card instead of eject when it prompt.
  2. Download testdisk using command brew install testdisk
  3. Restart your mac.
  4. Use command sudo photorec, must use sudo as administrator for detecting all disks.
  5. Select the broken SD card disk, you can unplug and plug it back in to see which disk is the SD card.
  6. Choose a distination folder, eg: ~/Desktop/recovery
  7. Click Search, it will start scan sectors, and save the images in distination folder.
  8. Wait till its finished…… sd card is slow……
  9. It may create multiple folders. Please look at result carefully.
