Detect exfat disk in mac

Cannot detect exfat external drive in my macbook. Here the solution, diskutil list get the identifer of the disk, eg: disk2s2 ps -ax | grep <identifier> get the proccess id of fsck sudo kill -9 <process id> Alert pops up and I can access the disk in finder.
